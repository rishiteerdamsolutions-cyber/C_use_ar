⚡ V1 — Fast Mode (Training Only)

  • Zero AI API calls at runtime
  • Uses saved pixel coordinates from training
  • Works completely offline
  • Best: same UI every run, high volume
  • Cost: ₹0 per run

🤖 V2 — AI Mode (Training + Claude)

  • Claude Vision re-checks every step live
  • Adapts if UI changes between runs
  • Never clicks wrong place after UI update
  • Best: new deploys, changing platforms
  • Cost: ~₹4 per run (Claude API)

🖥 Terminal Commands

Open your terminal in the agent folder and run:

# Run in AI mode (smart)
python main.py --run-workflow deploy_vercel --mode smart
# Run in Fast mode (no API)
python main.py --run-workflow deploy_vercel --mode fast
# Run with variable substitution
python main.py --run-workflow deploy_vercel --mode smart --var CLIENT_NAME=PriyaSalon
# Teach mode (interactive CLI)
python main.py --teach

📖 Quick Start Guide

Step 1 — First-time setup

pip install -r requirements.txt
python main.py --setup    # enter API keys into keychain
python main.py --check    # verify everything is ready

Step 2 — Teach the agent a workflow

Take screenshots of every step you want the agent to learn. Name them 1.png, 2.png, 3.png... Then:

# Option A: Use this dashboard — Teach tab above

# Option B: Copy screenshots to the input folder
cp *.png autonomous-web-agency/teach/screenshots_input/
python main.py --teach

Step 3 — Run the workflow

python main.py --run-workflow deploy_vercel --mode fast
python main.py --run-workflow deploy_vercel --mode smart

💡 Pro Tip — Variables

When teaching, use {{CLIENT_NAME}} or {{SITE_URL}} as placeholders in your step instructions. At runtime, pass real values:

python main.py --run-workflow deploy_vercel --var CLIENT_NAME=PriyaSalon --var SITE_URL=priya-beauty
