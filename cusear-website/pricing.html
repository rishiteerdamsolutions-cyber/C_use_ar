Frequently Asked Questions

Do I have to build or “teach” the agent myself?

No. Cusear™ is sold with a pre-trained ar™. You install, connect accounts, and run.

Do I need to code?

No. The desktop app is designed for regular users.

Does my computer need to be on all the time?

No 24/7 requirement. You choose a time window (for example 5–8 PM). The machine only needs to be awake and online during that window when a run is scheduled — same as using the apps yourself. If it’s off, the job waits and we’ll notify you on WhatsApp.

Device Requirements?

Mac or Windows laptop/desktop. Internet connection. Device must be ON during your set time window when posting is scheduled.

Where are prices in USD?

Billing is in INR. For reference, see the approximate USD line above the plans (~1 USD = ₹95).
