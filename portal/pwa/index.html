Your agent token

Used only on your computer. Never share it publicly.

macOS

  1. Download mac.command and agent-token.txt into the same folder.
  2. First time: right-click mac.command → Open → Open (bypasses Gatekeeper without a developer account).
  3. Clone repo to ~/cusear-agent, install requirements.txt, leave the terminal window open.
Download Mac launcher

Windows

  1. Download windows-run-agent.bat with agent-token.txt in the same folder.
  2. If SmartScreen appears: More info → Run anyway (unsigned script, not a signed installer).
  3. Clone repo to %USERPROFILE%\cusear-agent and pip install -r requirements.txt.
Download Windows launcher
