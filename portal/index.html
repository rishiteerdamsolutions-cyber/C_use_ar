How it works

Four steps from client request to live website — fully automated.

API

Dead simple to integrate

One API call to build and deploy a complete website.

import requests

# Build a salon website for a new client
response = requests.post(
    "https://api.yourplatform.com/api/v1/templates/build",
    headers={"X-API-Key": "ak_live_your_key_here"},
    json={
        "template_id": "salon",
        "client_name": "Priya Beauty Studio",
        "client_phone": "9876543210",
        "city": "Bangalore",
    }
)

print(response.json()["live_url"])
# → https://priya-beauty-studio.vercel.app

FAQ

Frequently asked questions

Everything you need to know before getting started.

Do I need coding skills to use the Desktop Agent?
No. You teach the agent by taking numbered screenshots of your screen and describing what each step does. No code required.
What is the difference between V1 Fast and V2 Smart?
V1 uses saved pixel coordinates — zero API cost, instant replay. V2 uses Claude Vision to re-locate elements on screen each run — adaptive to UI changes but costs ~₹5 per step.
Which payment methods are supported?
We use Razorpay — UPI, credit/debit cards, net banking, and wallets are all accepted. Plans are in Indian Rupees (INR).
Can I use my own domain for the white-label portal?
Yes. Point a CNAME from any domain (e.g. app.youragency.in) to yourplatform.com and set it in your Admin → Domains panel. No extra cost.
What happens when credits run out?
API calls return a 402 error. You can top up instantly via Razorpay from your dashboard or the API. No workflows are lost.
Is my API key secure?
API keys are shown only once and stored as SHA-256 hashes on our servers. We never log raw keys. Use your OS keychain locally.
