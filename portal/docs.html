Quick Start

Get from zero to a live website URL in under 5 minutes.

Billing is moving to Cashfree for India (UPI, cards, net banking). Some request fields and code paths may still use the legacy razorpay_* names until the server is fully migrated—treat them as the payment reference from your hosted checkout.

1. Get your API key

Sign up at signup.html, pick a plan from the pricing list, and complete secure checkout. Your API key is shown once—save it.

2. Make your first call

Build a complete salon website with one API call:

Python
JavaScript

Authentication

Every API request must include your API key in the request header.

Header format

X-API-Key: ak_live_xxxxxxxxxxxxxxxx

API keys are prefixed ak_live_ for production and ak_test_ for test mode. Keys are shown once at creation — store them securely in environment variables.

Security: Never embed API keys in client-side JavaScript or commit them to version control. Use server-side proxying for web applications.

Teaching Workflows

Teach the agent a new automation by uploading numbered screenshots of each step.

Step 1 — Record your workflow

Perform the workflow manually while taking screenshots at each key step. Name them 1.png, 2.png, 3.png, and so on.

Step 2 — Upload via API

import requests

files = [("screenshots", open(f"{i}.png", "rb")) for i in range(1, 6)]
data = {
  "workflow_name": "salon_build",
  "instructions": json.dumps([
    {"step": 1, "description": "Open Cursor AI"},
    {"step": 2, "description": "Click new project"},
    {"step": 3, "description": "Type the prompt in AI panel"},
    {"step": 4, "description": "Run npm dev"},
    {"step": 5, "description": "Deploy to Vercel"},
  ])
}
requests.post(
  "https://api.yourplatform.com/api/v1/workflows/teach",
  headers={"X-API-Key": "ak_live_YOUR_KEY"},
  data=data, files=files
)

Step 3 — Run with variables

requests.post(".../workflows/run", json={
  "workflow_name": "salon_build",
  "variables": {"CLIENT_NAME": "Riya Salon", "CITY": "Mumbai"}
})

Webhooks

The billing webhook endpoint receives payment events from Razorpay.

POST /api/v1/billing/webhook Razorpay only · Hidden from docs

Configure in Razorpay Dashboard → Webhooks → https://api.yourplatform.com/api/v1/billing/webhook

The endpoint verifies the HMAC-SHA256 signature using RAZORPAY_WEBHOOK_SECRET and auto-credits the wallet on payment.captured events.

Webhook signature verification

import hmac, hashlib

def verify(raw_body, signature, secret):
  expected = hmac.new(
    secret.encode(), raw_body, hashlib.sha256
  ).hexdigest()
  return hmac.compare_digest(expected, signature)
