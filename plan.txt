  PRODUCT 2 — Public REST API             STATUS: 🔲 Day 2 (TODAY)
  ─────────────────────────────────────────────────────────────────
  Anyone can use our agent engine via API key in their own software.
  FastAPI server. MongoDB for keys + usage. Razorpay wallet top-up.

  Endpoints:
    POST  /api/v1/run-workflow       run a saved workflow
    POST  /api/v1/teach              upload screenshots → workflow JSON
    POST  /api/v1/validate-prompt    GPT↔Gemini loop
    GET   /api/v1/templates          list website templates
    POST  /api/v1/build-website      full pipeline → live URL
    POST  /api/v1/keys/generate      create API key
    GET   /api/v1/keys/usage         calls used, credits, cost

  Pricing (Pay-per-call):
    /run-workflow    V2 AI mode   → ₹5 per call
    /run-workflow    V1 fast mode → ₹2 per call
    /teach           per screenshot analysed → ₹3
    /validate-prompt → ₹2 per call
    /build-website   full delivery → ₹50 per call

  Credit packs (Razorpay):
    ₹999  = 500 credits
    ₹2999 = 2000 credits
    ₹9999 = 10000 credits

  Tech stack:
    FastAPI + Uvicorn
    MongoDB Atlas (keys, usage, billing records)
    Razorpay (wallet top-up + webhook)
    Redis (rate limiting per key)
    Docker (deploy on DigitalOcean ₹500/mo VPS)

  Files to build:
    api/server.py          FastAPI app, CORS, middleware
    api/keys.py            generate, validate, hash API keys
    api/usage.py           track calls, deduct credits
    api/billing.py         Razorpay wallet + webhook handler
    api/middleware.py      auth, rate limit, request logging
    api/routes/
      workflows.py         /run-workflow, /teach
      templates.py         /templates, /build-website
      validator.py         /validate-prompt
      keys.py              /keys/generate, /keys/usage
    api/models.py          Pydantic request/response models
    api/database.py        MongoDB connection singleton
    api/rate_limiter.py    Redis-based rate limiting
    Dockerfile
    docker-compose.yml

  SECURITY NOTES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  - All API keys hashed with SHA-256 before storing in MongoDB
  - Keys prefixed: ak_live_xxxxx (production), ak_test_xxxxx (sandbox)
  - Rate limiting: Redis, per key, per minute + per day
  - Razorpay webhook signature verified before crediting wallet
  - No credentials ever in logs (CredentialRedactFilter active)
  - pyautogui FAILSAFE=True always (move mouse top-left to stop)
  - AES-256 local credential cache
  - OS keychain for all secrets (never plaintext on disk)
