Scheduled automation — run # window (optional)

For runs started by the automation scheduler, each run has a counter: 1st scheduled run = 1, 2nd = 2, … (CURRENT_AUTOMATION_RUN). Leave both fields empty to always run this step. Set First run # and/or Last run # so the step only runs inside that inclusive window — e.g. right-arrow only on runs 1–6, down-arrow only on run 7, left on 8–12, etc. Manual Run ignores these fields (every step always runs).

Snake grid navigation (single step)

Uses the scheduled automation run index (CURRENT_AUTOMATION_RUN: 1, 2, 3…). With defaults 6×5: run 1 → 1×Right; run 6 → 6×Right; run 7 → 6×Right + Down; run 8 → +1×Left; … run 13 → +Down; runs 14–18 add 1…5×Right on the next row; same boustrophedon pattern through run 30. Manual Run uses run 1 for this step (so you see minimal arrows). Optional run # window above still applies on scheduled runs only.

Run must copy the model’s reply to the system clipboard immediately before this step.

Hotkey keys (PyAutoGUI names, 1–6 keys)

Typical shortcuts use 2–4 keys; this form allows up to 6 (e.g. several modifiers + a letter). Order matters: list modifiers first, then the main key (same as pyautogui.hotkey). Examples: shift, command, o · ctrl, shift, p · command, space. Aliases: cmd → command, opt → option, meta → command.

Best AI™ — Overview

Best AI™ is a Trainer workflow pattern: one saved question, the same prompt bundle pasted on ChatGPT, Gemini, and Claude, replies pasted back here, then an OpenAI synthesizer on the Trainer host produces one final answer with clear reasoning (or a deliberate merge, or exclusion of a weak reply).

What it is
You drive the browsers (manually or with your own Saved Workflow steps). The Trainer UI holds the canonical query, collects each model’s reply in labeled slots, and calls OpenAI once to judge or merge them.
How it works (end-to-end)
  1. Save your question here (or align with CURRENT_TOPIC from an ar™ bundle).
  2. Copy the bundled prompt (query + formatting instructions) into each platform; bring each answer back into the matching slot.
  3. When ChatGPT, Gemini, and Claude slots are all non-empty, the synthesizer runs automatically after a brief debounce.
  4. Synthesizer uses OPENAI_API_KEY on the machine running dashboard.py and shows the final Best AI result in step 3.
All 10 platforms (roadmap)
ChatGPT, Gemini, Claude, Grok, Perplexity, DeepSeek, Copilot, Le Chat, Meta AI, You.com — Core flow uses the first three; extra rows are optional for synthesis if pasted.
